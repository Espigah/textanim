Property detail
indexproperty
public var index:int

Index works like a ID block in the TextAnim instance. Each block has a index, starting from 0 ascending to right. It serves as a reference for actions, delays and whatever it needs.

posXproperty 
public var posX:Number = 0

The original X position of the block, very useful for creating animations, this value is generated after all blocks are created and positioned. With this you always know where the original X position of text.

posYproperty 
public var posY:Number = 0

The original Y position of the block, very useful for creating animations, this value is generated after all blocks are created and positioned. With this you always know where the original Y position of text.
